# Neural Paper Mode

`paper-neural-learning` and `paper-neural-demo-spot` enable neural replay, continuous learning, self tuning, transformer/sequence challengers, committee, and RL execution flags only in paper mode.

Live neural autonomy remains disabled:

```env
NEURAL_LIVE_AUTONOMY_ENABLED=false
NEURAL_LIVE_AUTONOMY_ACKNOWLEDGED=
LIVE_TRADING_ACKNOWLEDGED=
```
